Content Calendar

A fully editable, browser-based content calendar for managing Instagram and Facebook posts. No login required. No app to install. Just open the link and start editing.

---

LIVE PAGE: https://mmmedea.github.io/Content-Calendar/

## ✨ What You Can Do

Every post card is fully editable directly in the browser:

| Field | How to Edit |
|---|---|
| 📅 Date | Click the date field — a date picker opens |
| 📝 Caption | Click and type directly on the card |
| 💬 CTA | Edit inline inside the green box |
| # Hashtags | Click and type |
| 🖼 Image | Click the image slot to upload from your device |
| 📱 Platforms | Toggle checkboxes (Instagram, Facebook, LinkedIn, Twitter) |
| 🎨 Format | Dropdown — Carousel, Static Image, Reel, Story, Video |
| 🏷 Status / Legend | Dropdown — changes the card color and moves it between sections |
| 🎯 Theme | Dropdown — Educational, Social Proof, Product/Brand, etc. |
| 💡 Notes | Type in the comment field at the bottom of each card |

---

## 📊 Auto-Updating Trackers

The sidebar updates **instantly** as you make changes:

- **Platform Tracker** — counts posts per platform based on your checkbox selections
- **Status Tracker** — counts posts per status (Ideation, In-Progress, Ready for Review, etc.)
- **Header chips** — shows total posts, active platforms, and ready count at a glance

---

## ➕ Adding & Removing Posts

- Click **+ Add Post** in the sidebar to create a new card
- Fill in the date, platforms, format, status, and caption in the popup
- Click the **✕** on any card to delete it

## 🛠 How to Update the Calendar

1. Make your edits in the browser
2. When you want to save a new version permanently:
   - Open `index.html` in a text editor (Notepad, VS Code, etc.)
   - Update the `posts` array at the top of the `<script>` section
   - Upload the updated file back to GitHub → the live site updates automatically

## 🎨 Status Color Guide

| Status | Color | Meaning |
|---|---|---|
| 🟡 Ideation | Amber | Post idea — caption not written yet |
