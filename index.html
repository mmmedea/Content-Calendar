🔗 Admin Setup — Firebase Sync

You are the admin. Paste your Firebase Realtime Database URL below. Once saved, every device that opens this GitHub link will sync automatically — no setup needed on their end.

1. Go to console.firebase.google.com → New project
2. Click Realtime Database → Create database → Test mode
3. Go to Rules tab → paste: {"rules":{".read":true,".write":true}} → Publish
4. Copy the DB URL (e.g. https://your-app-rtdb.firebaseio.com)
5. Paste below, click Save, then re-upload this file to GitHub

After clicking Save, download the updated file and push it to your GitHub repo. Everyone who opens your GitHub Pages link will then sync automatically — they will never see this screen.
